
Messaging system - rest API.
----------------------------------------------------------
A message contains :

1. Sender (owner)
2. Receiver
3. Message
4. Subject
5. Creation date

------------------------------------------------------------
The rest API methods :

- Write message
- Get all messages for a specific user
- Get all unread messages for a specific user
- Read message (return one message)
- Delete message (as owner or as receiver)
------------------------------------------------------------
